    Challenges and Solutions

    • Challenge1 : Because proximity touch sensor was very small and noise was injected from hardware pcb, sensitivity was very poor and noise caused false function.
      Solution1 : I increased software gain to improve sensitivity and implemented filter combination algorithm(Median + Kalman) to suppress noise which casued by gain. And also I developed pre-emphasis algorithm which have different weight according to specified range to speed up response time.

    • Challenge2 : When I communicated with 8bit-microprocessor through I2C, there was stop issue of main processing, for this one provide only 1 interrupt vector and I2C interrupt disturbed mian ADC processing.
      Solution2 : Checked free time between end and restart of of ADC processing. This time was indicated by IO port which connected between two processors. I got I2C communication of 1 byte to perform only during this time.
